## Summary
To authenticate with the API, you must generate an API key from the dashboard and include it in the Authorization header of your requests. API keys expire after 90 days.

## Relevant Endpoint
N/A

## Description
Authentication is performed by including your API key in the `Authorization` header using the `Bearer` authentication scheme. You can generate your key by logging into the dashboard, navigating to **Settings**, and selecting **API Keys**.

## Parameters
No parameters

## Request Example
```python
# Python
headers = {
    "Authorization": "Bearer YOUR_API_KEY"
}
```

```javascript
// JavaScript
const headers = {
    "Authorization": "Bearer YOUR_API_KEY"
};
```

```bash
# cURL
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.example.com/endpoint
```

## Important Notes
- API keys expire after 90 days.
- Using an invalid or expired key will result in a `401 Unauthorized` error.
- Ensure your account has sufficient permissions, as a `403 Forbidden` error may occur if permissions are lacking.
