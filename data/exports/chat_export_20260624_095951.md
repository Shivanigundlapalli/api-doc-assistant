### User
How do i authenticate?

### Assistant
API authentication requires an API key (Source: Authentication Guide).

To set up authentication:
1.  **Generate an API Key**:
    *   Log in to the dashboard.
    *   Navigate to Settings.
    *   Click API Keys.
    *   Generate a new API key.
    (Source: Generating an API Key)
2.  **Use the API Key**:
    *   Include the key in the `Authorization` header in the format: `Authorization: Bearer YOUR_API_KEY`.
    (Source: Using the API Key)
